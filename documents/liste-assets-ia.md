# Liste des assets générés par IA — Projet Cali

## Outil utilisé

**Python Pillow (PIL) v10+** — bibliothèque de traitement d'image Python, utilisée ici pour générer programmatiquement des illustrations schématiques à partir de formes géométriques simples (cercles, lignes, polygones).

> Les images ont été produites via un script Python (`gen_images.py`) piloté par Claude (Anthropic) : le modèle a conçu et écrit l'intégralité du code de génération, défini les coordonnées des silhouettes, les couleurs et les flèches directionnelles pour chaque exercice.

**Spécifications appliquées :**
- Format : PNG 400 × 400 px
- Style : silhouette en trait noir sur fond blanc
- Couleur d'accent : rouge uniquement (flèches de direction)
- Aucun texte dans les images
